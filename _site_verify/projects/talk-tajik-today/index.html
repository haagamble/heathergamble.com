Talk Tajik Today

Live- Website

Overview

Talk Tajik Today is a language-learning website designed to help learners find resources. It includes language-learning tips, downloadable materials, and links to useful Tajik content online.

I first launched the site in 2013 after discovering how few online resources existed for learning Tajik.

How It Started

The idea began during conversations with other language coaches. We were wondering whether there were any good websites for learning Tajik. At the time, there were very few.

Someone asked if anyone knew how to build a website. No one did.

So I decided to learn.

I took a course in HTML and CSS and built the first version of Talk Tajik Today. The original site was a simple static website with curated links and downloadable learning materials.

How the Project Evolved

Over the years, the site has gone through several versions.

The first version was a basic HTML and CSS site that organized Tajik learning resources and included downloadable PDFs.

Later I rebuilt the site using WordPress so I could add more functionality and publish content more easily.

For about three years I also ran a Word-A-Day email for subscribers. Each email included:

  • a Tajik word
  • example sentences
  • related vocabulary
  • notes about usage, prefixes, or word origins

The Current Version

A few years ago I stopped paying for hosting and moved the site to GitHub Pages. The site is now a static website again, but with many more links and curated resources.

I have also added some interactive resources, including Tajik Wordle and activities designed to help learners practice the first 1,000 words.

What I Learned

This project taught me how to build and maintain a website from scratch. It also showed me how useful well-organized resources can be for learners of a language that does not have many online materials.

Over time the project also pushed me to learn new tools, including GitHub, WordPress and Netlify.
