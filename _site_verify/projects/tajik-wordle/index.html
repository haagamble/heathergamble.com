Overview

Tajik Wordle is a Wordle game in Tajik. I made it in Feb 2022 at the peak of the worldwide Wordle craze.

What I Learned

This was the first time I forked someone else's code and adapted it into a project of my own. Building this project helped me understand how to work with an existing codebase and modify it.

Challenges

The main challenge was building the dictionary of 5-letter Tajik words.
