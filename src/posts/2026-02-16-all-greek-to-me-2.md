I love finding words that are easy to learn. Here are some words that **sound similar enough in Greek that English speakers can easily recognize them**.

### Music and the Arts

| Greek  | English |
| ------ | ------- |
| κιθάρα | guitar  |
| πιάνο  | piano   |
| βιολί  | violin  |
| κάμερα | camera  |
| σκηνή  | scene   |

### Abstract and Academic Words

| Greek      | English   |
| ---------- | --------- |
| θεωρία     | theory    |
| ανάλυση    | analysis  |
| διάλογος   | dialogue  |
| κρίση      | crisis    |
| χαρακτήρας | character |
| σύμβολο    | symbol    |
| τύπος      | type      |
| τόνος      | tone      |
| φάση       | phase     |

These words are not identical to English, but the similarity often makes them easier to remember.

Happy language learning!
