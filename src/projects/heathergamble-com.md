## Overview

HeatherGamble.com is my personal website where I share what I am learning, the projects I am building, and some of the topics that interest me. It serves as a place to document my experiments with language learning, web development, and AI tools.

I purchased this domain in 2025 and began building the site in 2026.

## Why I Built This Site

This site is meant to function as both a personal website and a learning journal.

## How the Site Is Built

The current version of the site is built using Eleventy, a static site generator. I chose Eleventy because it allows me to keep the site simple while still organizing content into sections such as:

- my language-learning diary
- articles and guides
- project pages
- personal notes and experiments

While building the site I have been experimenting with several AI tools, including ChatGPT, GitHub Copilot, and Codex, which I use as coding assistants while developing new features.

## What I Learned

Building this site has helped me better understand how static sites work, including templating, collections, layouts, and build processes. It has also been an opportunity to experiment with modern development tools.
